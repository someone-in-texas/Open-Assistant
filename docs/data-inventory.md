# Data inventory

| Data                                                           | Source            | Purpose/destination                      | Retention                                       | Basis                   | Firefox category                     | Required                         | Deletion/logging                                         |
| -------------------------------------------------------------- | ----------------- | ---------------------------------------- | ----------------------------------------------- | ----------------------- | ------------------------------------ | -------------------------------- | -------------------------------------------------------- |
| Page excerpt, title, URL, locator                              | User-approved tab | Ground response; configured relay/OpenAI | Session locally; zero application-log retention | Explicit feature action | `websiteContent`, `browsingActivity` | Required for contextual features | Remove chip/restart/account deletion; not logged         |
| Prompt and response                                            | User/AI           | Fulfill request; relay/OpenAI            | Messages local; zero application-log retention  | Explicit submission     | `personalCommunications`             | Required for chat                | Delete/export locally; not logged                        |
| OIDC subject/access token                                      | Identity provider | Relay access control                     | Access token memory; server account per policy  | Sign-in                 | `authenticationInfo`                 | Required in hosted mode          | Sign out/account deletion; authorization header redacted |
| Agent observation/action metadata                              | Authorized tab    | Optional bounded agent                   | Context session; audit metadata up to 7 days    | Separate opt-in         | `websiteActivity`                    | Optional                         | Stop/delete/retention expiry; no raw page log            |
| Version, OS family, Firefox major, coarse success/timing/error | Extension runtime | Optional product telemetry               | Documented aggregate window                     | Separate opt-in         | `technicalAndInteraction`            | Optional                         | Disable/delete; no content/URL/user ID                   |

The manifest declarations are conservative because contextual use necessarily transmits the listed required categories. `websiteActivity` and `technicalAndInteraction` remain optional and feature-gated. The automated consistency check verifies these labels and defaults.
