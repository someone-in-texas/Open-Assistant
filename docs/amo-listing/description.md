# Open Assistant for Firefox

Open Assistant gives you a sidebar for asking questions about the current page, selected text, or tabs you explicitly choose. It shows every source before transmission, streams answers with source references, previews writing changes as a diff, and applies them only when you click Apply. One-step undo is included for supported editors.

Mock mode works locally without a paid API account. Hosted mode requires the published service account; self-hosted and optional native BYOK modes are documented for technical users. Service/API costs may apply outside mock mode.

The optional ChatGPT tab handoff inserts a reviewed context bundle into the visible composer and never sends it. The project does not read ChatGPT cookies, prior conversations, memories, files, or network traffic.

Page content is processed only when you invoke an AI feature and send a request. Agent mode is experimental and interaction is disabled pending an independent security gate. This project is not affiliated with or endorsed by OpenAI.
